Deploy to GitHub Pages in 5 Minutes

Host this entire site for free on GitHub Pages with your own custom domain

1

Create a GitHub Repository

Go to github.com/new and create a new repository. Name it socialmediahouse (or any name you prefer). Make it Public so GitHub Pages can serve it.

2

Upload This File

Click "Add file" > "Upload files" on your repository page. Upload this index.html file. You can also use Git from the command line:

git clone https://github.com/YOUR_USERNAME/socialmediahouse.git cd socialmediahouse # Copy index.html here git add index.html git commit -m "Initial commit" git push origin main
3

Enable GitHub Pages

Go to your repository Settings > Pages. Under "Source", select main branch and / (root) folder. Click Save. Your site will be live at YOUR_USERNAME.github.io/socialmediahouse within 1-2 minutes.

4

Add Custom Domain (Optional)

In Settings > Pages, under "Custom domain", enter your domain like socialmediahouse.online. Then add a CNAME record in your DNS settings pointing to YOUR_USERNAME.github.io. Enable "Enforce HTTPS".

# DNS Configuration (at your domain registrar) Type: CNAME Name: @ (or www) Value: YOUR_USERNAME.github.io # Or if using A records: Type A Name: @ Value: 185.199.108.153 Type A Name: @ Value: 185.199.109.153 Type A Name: @ Value: 185.199.110.153 Type A Name: @ Value: 185.199.111.153
5

Add a CNAME File (If Using Custom Domain)

Create a file named CNAME (no extension) in the same directory as index.html with just your domain inside:

socialmediahouse.online

Commit and push this file. GitHub Pages will automatically configure SSL for your custom domain.

Frequently Asked Questions

Are all Social Media House tools really free?

Yes, all 70+ tools on Social Media House are 100% free to use with no hidden charges, subscriptions, or credit card requirements. You can use every tool unlimited times. The only optional component is the Google Gemini API key for the AI Agent, which has a generous free tier.

Which platforms does Social Media House support?

Social Media House supports Instagram, YouTube, TikTok, Facebook, LinkedIn, X (Twitter), Pinterest, plus general-purpose tools that work across all platforms including content calendars, engagement boosters, and analytics summaries.

How does the AI Agent work?

The AI Agent uses Google Gemini AI to analyze your input and generate optimized content. Click the gear icon in the AI sidebar, paste your free Gemini API key, and save. Your key is stored only in your browser's local storage and never sent to any third-party server.

Do I need to create an account?

No account is needed. All tools are accessible instantly without sign-up. Just visit the site and start using any tool immediately. The AI Agent's API key is saved locally in your browser.

How do I deploy this on GitHub Pages?

Create a GitHub repository, upload this index.html file, go to Settings > Pages, select the main branch, and your site will be live at username.github.io/repo-name within minutes. See the detailed deployment guide above for step-by-step instructions including custom domain setup.

Can I customize the logo and branding?

Yes! Click on the "SMH" logo placeholder in the navbar to upload your own custom logo image. The logo is stored in your browser's local storage and persists across sessions. You can also edit the HTML to change the site name, colors, and all text content.

Is my API key safe?

Your Google Gemini API key is stored only in your browser's local storage and is never sent to any server other than Google's Gemini API directly. It is never shared, logged, or transmitted to third parties. For additional security, you can revoke and regenerate your key at any time from the Google AI Studio dashboard.
